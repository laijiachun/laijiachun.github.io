In order to ensure the realization of the relevant functions of "This App" and the safe and stable operation of the application, we will access the software development kit (SDK) provided by a third party to achieve related purposes according to the needs of actual business scenarios.

We will conduct strict security monitoring on the software tool development kits (SDKs) used by partners to obtain information to protect data security.

The relevant third-party SDKs we access to are listed in the catalog.

Please note that the third-party SDK may change the data processing type due to its version upgrade, policy adjustment, etc., please refer to the official description of its formula.

 

1. Umeng SDK

Purpose of use: to provide statistical analysis services and basic anti-cheating capabilities

Get information: device MAC address, unique device identifier (IMEI/OPENUDID/GUID), advertising identifier, SIM card IMSI information, network status, WiFi status, WiFi MAC address

Invoked device permissions: get network status, access WiFi status, allow applications to network

Privacy Policy: https://www.umeng.com/policy

 

2. Pangolin SDK

Purpose of use: advertising cooperation, advertising attribution, anti-fraud

getting information:

Equipment information: basic information such as equipment brand, model, software system version, etc.

Application information: developer application name, application package name, version number, application installation list and other information

Other information: operator information, device time zone, device language, wireless network SSID name, WiFi router MAC address, device MAC address, IP address, device identifier (such as IMEI, OAID, IMSI, ICCID, GAID, MEID, specific fields There are differences due to different software and hardware versions)

Invoked device permissions: get network status, access WiFi status, allow applications to network

Privacy Policy: https://www.pangle.cn/privacy/partner

 

3. AdMob SDK

Purpose of use: push advertising information to users, anti-cheating

Get information: unique identifier, device type and settings, operating system, mobile network information (including carrier name) and app version numbers, IP addresses, crash reports

Called device permissions: get network status, access WiFi status

Privacy Policy: https://policies.google.cn/privacy
